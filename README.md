## command

* npm init

* npm install gulp -g & npm install --save-dev gulp-util

* npm install --save-dev gulp-babel@7.0.1 babel-cli babel-core babel-preset-es2015 // 安装 babel 模块, 支持ES5语法

* npm run dev // [gulp]

## run

* npm run dev   # 启动项目

* npm run clean # 清理项目
